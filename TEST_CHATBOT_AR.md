# اختبار الشات بوت - دليل سريع

## 🚀 قبل الاختبار

### 1. تشغيل Backend
```bash
cd backend
npm run dev
```
✅ انتظر حتى ترى: `Server running on port 5000`

### 2. تشغيل Frontend
```bash
cd frontend/Web
npm start
```
✅ سيفتح المتصفح تلقائياً

## ✅ علامات النجاح

الشات بوت يعمل بشكل صحيح إذا:
- ✅ يرد على الترحيب بشكل ودي
- ✅ يفهم الأسئلة بالإنجليزية والعربية
- ✅ يعطي رسائل واضحة عند عدم الفهم
- ✅ يعرض معلومات مفصلة عن الحضور والكورسات
- ✅ يرد على الشكر والوداع بشكل طبيعي
- ✅ لا يعرض رسائل خطأ عامة مثل "I encountered an error"

## ❌ علامات المشاكل

إذا رأيت:
- ❌ "I encountered an error. Please try again."
- ❌ "Connection Error"
- ❌ رسائل خطأ غير واضحة
- ❌ لا يرد على الرسائل

**الحل:**
1. تأكد من أن Backend يعمل
2. تأكد من أن Frontend يعمل
3. افتح Developer Tools (F12) وتحقق من Console
4. تحقق من Network tab في Developer Tools

## 🔧 إصلاح المشاكل

### المشكلة: "Connection Error"
**السبب:** Backend لا يعمل
**الحل:**
```bash
cd backend
npm run dev
```

### المشكلة: "User not found"
**السبب:** المستخدم غير موجود في Firebase
**الحل:** تحقق من Firebase Console

### المشكلة: "Role not assigned"
**السبب:** المستخدم بدون role
**الحل:** أضف role في Firebase:
```javascript
{
  name: "Student Name",
  email: "student@example.com",
  role: "student"
}
```

### المشكلة: لا يرد على الرسائل
**السبب:** خطأ في الكود أو Backend
**الحل:**
1. تحقق من Console في المتصفح (F12)
2. تحقق من terminal الـ Backend
3. أعد تشغيل Backend
